# PPA Captures

Real-world packet captures from "Practical Packet Analysis, 3rd Edition" by Chris Sanders.

These captures are NOT committed to git (110MB total). Download them from:

**https://nostarch.com/packetanalysis3/**

## Usage

Selected high-value captures are trimmed and copied to `fixtures/smoke/` with a `ppa-` prefix for integration testing. See `fixtures/smoke/PROVENANCE.md` for details.

## Available Captures

65 pcapng files covering: DNS, TCP, HTTP, ICMP, ARP, TLS, IPv6, DHCP, 802.11, security scenarios (SYN scan, ARP poisoning, malware C2, session hijacking, RAT infection).
